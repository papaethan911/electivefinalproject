# Personal Portfolio Website

This is a responsive personal portfolio website built as a final project for IT ELECTIVE 4. It showcases my skills, projects, and provides a way to contact me.

## Technology Stack

- HTML5 (Semantic elements)
- CSS3 (Custom styles with Flexbox and CSS Grid)
- JavaScript (for interactive elements)
- Git/GitHub (for version control and deployment)

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Semantic HTML**: Uses proper HTML5 tags for accessibility
- **CSS Variables**: For easy theming and maintenance
- **Interactive Navigation**: Hamburger menu for mobile devices
- **Sections**:
  - Hero: Introduction with call-to-action
  - About Me: Biography and profile photo
  - Skills: Visual representation with progress bars
  - Projects: Gallery of at least 3 projects
  - Contact: Form and social links

## Deployment

This website is deployed using GitHub Pages. To deploy your own version:

1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to Settings > Pages
4. Select the main branch as source
5. The site will be live at `https://yourusername.github.io/repository-name/`

## How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser

No additional setup required as it's a static website.

## Customization

- Update the content in `index.html` with your personal information
- Modify styles in `styles.css` to match your preferences
- Replace placeholder images with your actual photos and project screenshots
- Update links in the contact section with your social media profiles
