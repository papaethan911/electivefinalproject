# Personal Portfolio Website

This is a responsive personal portfolio website built as a final project for IT ELECTIVE 4. It showcases my skills, projects, and provides a way to contact me.

## Technology Stack

- HTML5 (Semantic elements)
- CSS3 (Custom styles with Flexbox and CSS Grid)
- JavaScript (for interactive elements)
- Git/GitHub (for version control and deployment)

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Semantic HTML**: Uses proper HTML5 tags for accessibility
- **CSS Variables**: For easy theming and maintenance
- **Interactive Navigation**: Hamburger menu for mobile devices
- **Sections**:
  - Hero: Introduction with call-to-action
  - About Me: Biography and profile photo
  - Skills: Visual representation with progress bars
  - Projects: Gallery of at least 3 projects
  - Contact: Form and social links
